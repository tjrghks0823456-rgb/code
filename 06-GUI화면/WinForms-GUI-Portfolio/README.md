# WinForms GUI Portfolio

C# Windows Forms GUI practice project.

## Screens

- Text / Login Demo (`TextLoginForm`)
- Number Guessing Game (`NumberGuessingForm`)
- Calculator (`CalculatorForm`)
- Todo List (`TodoListForm`)

## Environment

- Visual Studio 2022
- .NET Framework 4.7.2
- Windows Forms

## Run

Open `WinForms-GUI-Portfolio.sln` in Visual Studio and start debugging.

The app opens a launcher screen, and each button opens one GUI demo.
