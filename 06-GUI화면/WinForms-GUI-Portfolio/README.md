# WinForms GUI Portfolio

C# Windows Forms로 만든 GUI 연습 프로젝트입니다.

## Screens

- Text / Login Demo
- Number Guessing Game
- Calculator
- Todo List

## Environment

- Visual Studio 2022
- .NET Framework 4.7.2
- Windows Forms

## Run

Open `WinForms-GUI-Portfolio.sln` in Visual Studio and start debugging.

The app opens a launcher screen, and each button opens one GUI demo.
